# rename-config.txt -- mapping for rename-files.py
#
#   uv run rename-files.py -n rename-config.txt   # dry run (recommended first)
#   uv run rename-files.py    rename-config.txt   # apply
#
# Each line maps an old .ptx file to a new BASE name; the file stays in its
# current directory and its xml:id (plus every xi:include / xref that points at
# it) is updated to match. New names were chosen from each file's title and
# content. The dominant change is stripping the College Board "topic-N-N-"
# numbering, which this book intentionally does not mirror; CamelCase practice
# files are also normalized to lowercase kebab-case.
#
# Files whose current name is already a good content-based name are listed as
# comments ("# KEEP ...") so the survey is complete but they are not renamed.
